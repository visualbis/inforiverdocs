Export Visual Themes/Report Configs

In Inforiver visual you can export the custom report configuration data and visual theme as a JSON file format. A JSON file provides granular level customizations that can be easily shared and imported in a new report. Using the export custom theme/config options, you can standardize the reports and charts formats and visual configurations across all the reports.

Export Custom Theme/Configs#

  • Select the Advanced tab on the ribbon and select Export/Import.

Export/Import drop-down menu list

  • You can choose to export the visual theme settings or configurations.
    • Export Theme: Design changes (colors and formattings) settings applied in the visual.
    • Export Config: Visual configurations such as navigation, scaling, number format, conditional format, and templates.
  • Select the export type as "Export Theme" or "Export Config":

Exporting the visual's configurations

  • Right Click on the the .json file name and select Save Link As option.
  • Navigate to your desired storage location and select Save.

Your visual data will be downloaded in a json format.

JSON Theme File Format#

The most basic JSON file has following fields:

Field/Section NameDescription
typeType of data export. '1' is for theme file, and '2' for a config file.
chartSettingsVisual chart layout type and settings.
chartTitleTitle field settings
annotationAnnotation configurations
headerOptionTable Header settings
inforiverPropsInforiver global visual formatting configurations.
subTotalsRow and Column level sub totals setting to true or false.
exportExport settings such as export type, scaling, page size etc.
metadataRows and Columns metadata details.
