XOR

An XOR statement returns a logical 'Exclusive Or' of all arguments.

An XOR statement returns TRUE if one or more of the arguments evaluate to TRUE and returns FALSE in case all the arguments evaluates to TRUE. The XOR function returns FALSE when all the arguments evaluate to FALSE as well.

Syntax#

XOR ( logical_test1, [logical_test2], ... )

Return value#

Remark#

The result of XOR is TRUE when the number of TRUE inputs is odd and FALSE when the number of TRUE inputs is even.

Example#

Consider an investment scenario of investing in either stocks or property, but not both.

To achieve this, the formula should be:

IF (XOR (stocks, property), invested, not invested).
